## API: Google Sign-In Error

**Host:** `www.finsible.app`  
**Path:** `/auth/googleSignIn`  
**Method:** `POST`  
**Status:** `200 OK`

### Response Body

```json
{
  "message": "Could not log you in. Please try again.",
  "success": false,
  "data": null,
  "cache": false,
  "cacheTtlMinutes": 10000
}
```
